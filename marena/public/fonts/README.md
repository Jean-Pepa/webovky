# Font Vafle (BC Vafle)

Web je nastavený tak, aby používal font **Vafle**. Stačí sem vložit svůj
**licencovaný** soubor fontu a pojmenovat ho `vafle` — web ho hned začne používat
na všechny texty.

Podporované názvy (stačí jeden, ideálně woff2):

- `vafle.woff2`  ← nejlepší pro web
- `vafle.woff`
- `vafle.otf`
- `vafle.ttf`

Dokud tu soubor není, web použije systémový font (nic se nerozbije).

> BC Vafle je komerční font od Briefcase Type — použij prosím svou licenci
> (fakulta architektury / studio ho většinou má). Nestahuj ho z „free font"
> webů, bývají to nelegální kopie.
