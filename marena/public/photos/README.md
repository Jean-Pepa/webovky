# Fotky na homepage

Sem nahraj reálné fotky Mařeny (dvůr fakulty, bar, party, průvod, Fléda, křest…).
Dokud tu soubor není, web ukáže stylový placeholder s popiskem — nic se nerozbije.

Použij přesně tyto názvy (JPG, ideálně na šířku, klidně 1600 px a víc):

- `hero.jpg` — velká úvodní fotka (dvůr / nabitá party / atmosféra)
- `prednasky.jpg` — přednáška v aule
- `obedy.jpg` — obědy / snídaně na dvoře
- `bar.jpg` — bar na dvoře
- `party.jpg` — večerní koncert / DJ / dav
- `pruvod.jpg` — průvod městem (lidi v pytlích, maskot)
- `krest.jpg` — křest na Flédě

Tip: na hero vyber fotku, kde je hodně lidí a dobrá nálada — to prváky chytne nejvíc.
Dej pozor na práva k fotkám (ideálně vlastní z minulých ročníků).
